## 📗 プロジェクトの概要

Next.js ＆ AWS の学習用サンプルアプリケーションです。


## 🌐 Demo

![投稿一覧画面](./app1.png "投稿一覧画面")
![投稿画面](./app2.png "投稿画面")

- ログイン/ログアウト
- 会員登録
- 投稿一覧
- 投稿詳細
- マイページ（一覧・登録・更新・削除）

```

```
